İş Təcrübəsi

Mart 2022 — İndi
WSEC Azərbaycan
📍 Bakı
● Cari
Lead Software Engineer & Kibertəhlükəsizlik
  • wsec.edu.az təhsil platformasının sıfırdan arxitekturası, texniki rəhbərliyi və tam inkişaf dövrünün (SDLC) idarə edilməsi.
  • Tələbə qeydiyyat və müraciət sisteminin hazırlanması — çoxmərhələli iş axını, sənəd idarəetməsi və status izləmə.
  • Claude API və ChatGPT API inteqrasiyası əsasında AI chatbotun hazırlanması — WhatsApp və veb kanalları üzrə 3000+ istifadəçiyə avtomatik dəstək.
  • RESTful API arxitekturasının qurulması və mobil tətbiq inteqrasiyasının təmin edilməsi.
  • Saytın axtarış sistemlərində görünürlüyünün artırılması — texniki SEO optimallaşdırması, strukturlaşdırılmış məlumat işaretləməsi və səhifə sürətinin yüksəldilməsi.
  • Kibertəhlükəsizlik insidentlərinin idarəsi (Incident Response) — real vaxt monitorinq sistemi vasitəsilə zərərli IP-lərin avtomatik bloklanması, şübhəli trafik davranışlarının aşkarlanması və risklərin proaktiv şəkildə minimallaşdırılması.
  • 2025-ci ilin sentyabr ayında wsec.edu.az platformasına qarşı həyata keçirilən koordinasiyalı çoxmərhələli hücumun (database injection və autentifikasiya bypass cəhdləri daxil olmaqla) erkən mərhələdə aşkarlanması və aktiv müdaxilə ilə tam neytrallaşdırılması — verilənlər bazası və istifadəçi məlumatları tam qorunmuş, xidmət fasiləsizliyi təmin edilmişdir.
  • Yüksək həcmli məlumatlar üçün MySQL verilənlər bazasının optimallaşdırılması və inkişaf komandasının texniki rəhbərliyi.
İyun 2023 — İndi
Dendrologiya Bağı (Publik Hüquqi Şəxs)
📍 Bakı
● Cari
IT Mütəxəssisi & Sistem Administratoru
  • dendrologiya.gov.az domeninin rəsmi əldə edilməsi, saytın sıfırdan hazırlanması və Azərbaycan Xüsusi Rabitə və İnformasiya Texnologiyaları Agentliyinin (AzXRİTDA) apardığı pentest yoxlamaları nəticəsində aşkar edilmiş təhlükəsizlik boşluqlarının tam aradan qaldırılması.
  • Kompüter avadanlığının texniki xidməti — RAM, HDD/SSD, anakart dəyişiklikləri və yüksəltmələri.
  • İnstitut daxilində şəbəkə infrastrukturunun qurulması — internet kabellərin çəkilməsi, router/switch konfiqurasiyası.
  • Əməkdaşların kompüter, printer və periferik cihazlarla bağlı texniki problemlərinin həlli.
  • Əməliyyat sistemlərinin quraşdırılması, proqram yeniləmələri və antivirus idarəetməsi.
